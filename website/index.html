Quick Start: Add to Your AI Setup

Integrate Neo N3 MCP server with your AI assistant in just a few steps:


# Install the server
npm install @r3e/neo-n3-mcp

# Add this to your claude_desktop_config.json or MCP settings:
{
  "mcpServers": {
    "neo-n3": {
      "command": "npx",
      "args": [
        "-y",
        "@r3e/neo-n3-mcp"
      ]
    }
  }
}
