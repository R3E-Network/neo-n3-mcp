Quick Start

Get started with Neo N3 MCP server in just a few steps:


# Install the server
npm install @r3e/neo-n3-mcp

# Start the server
npx neo-n3-mcp

# Call an MCP tool
const result = await callTool('get_blockchain_info', {
  network: 'mainnet'
});
console.log(result);
